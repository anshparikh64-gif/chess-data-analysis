# Chess Data Analysis Project

This project analyzes a dataset of 13,992 chess games using Python and pandas.

## Features
- Calculated win rates (White vs Black vs Draw)
- Identified top 5 most common openings
- Analyzed rating differences and their impact on outcomes

## Tools Used
- Python
- pandas
- Excel

## Description
This project was created as part of my independent learning in data analysis and programming. It demonstrates my ability to work with real-world datasets, identify patterns, and extract insights.
